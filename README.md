# Captain Jack Website

**Modern Boat Storage & Smart Service** - Professionelle Webseite für Bootslagerung und Keelcrab-Technologie in Preveza, Griechenland.

🌐 **Live Preview:** https://captain-jack-preview.netlify.app  
📁 **GitHub Repo:** https://github.com/OBM-OPC/captain-jack-website

---

## Projekt-Übersicht

6-seitige responsive Website mit Fokus auf:
- Moderne Bootslagerung (Trockenlagerung, Stellplätze, Camper Parking)
- Innovative Keelcrab® Unterwasserreinigungstechnologie
- Unterwasser-Drohnen Inspektionen
- Digitale Buchungsprozesse

---

## Seitenstruktur

| Seite | Beschreibung |
|-------|--------------|
| `index.html` | Startseite mit Hero, USPs, Services-Übersicht |
| `storage.html` | Storage & Parking (Trockenlagerung, Stellplätze, Camper) |
| `service.html` | Bootsservice mit Keelcrab® Technologie |
| `technologie.html` | Technologie-Showcase (Roboter, Drohnen, Prozesse) |
| `ueber-uns.html` | Unternehmensprofil, Team, Standort Preveza |
| `kontakt.html` | Kontaktformulare + Google Maps Einbindung |

- **Headlines:** Poppins, 600-700

---

## Designentscheidungen

Die Website folgt einer klaren **Problem → Lösung → Vorteil → CTA** Struktur:

1. **Problem**: Traditionelle Bootspflege ist zeitaufwändig, teuer, gefährlich
2. **Lösung**: Keelcrab-Technologie ermöglicht Reinigung im Wasser
3. **Vorteil**: Kosteneinsparung, Zeiteinsparung, schonende Reinigung
4. **CTA**: Termin buchen / Anfrage senden

## Installation & Entwicklung

```bash
# Repository klonen
git clone https://github.com/OBM-OPC/captain-jack-website.git

# In das Projektverzeichnis wechseln
cd captain-jack-website

# Lokaler Server starten (Python)
python -m http.server 8001

# Oder mit Node.js
npx serve -l 8001
```

---

## Deployment

Die Website ist auf Netlify gehostet:

# Deploy auf Production
netlify deploy --prod --site=c5e8e9d1-8ed1-4e18-8138-4cb02a047a02
